== Frequently Asked Questions ==

= Where is plugin data stored? =
All settings are stored in the `wp_options` table under:
`hupuna_button_contact_settings`

= Does this plugin require Contact Form 7? =
No. CF7 is optional. You can use the plugin without CF7.

= Will plugin data be removed when deleting plugin? =
Yes, if `uninstall.php` exists, plugin data will be removed when you delete the plugin.

= Can I hide buttons on mobile or desktop? =
Yes. You can hide buttons on Mobile, Tablet, or Desktop from settings.
