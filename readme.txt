A professional contact button plugin for WordPress, including Hotline, Zalo, Telegram, and social media links.

== Description ==

This is a base plugin for managing contact buttons. 

== Installation ==

1. Upload the `hupuna-button-contact` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
