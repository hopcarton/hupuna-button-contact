== Installation ==

1.  Upload the plugin folder `hupuna-button-contact` to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Go to **Button Contact** menu to configure your buttons.
