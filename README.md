# Hupuna Button Contact

Base plugin for contact buttons (Hotline, Zalo, Tiktok, etc.) designed for internship projects.

## Project Structure
- `hupuna-button-contact.php`: Main entry point.
- `inc/class-hbc.php`: Core initialization.
- `inc/class-hbc-settings.php`: Admin settings.
- `inc/class-hbc-frontend.php`: Frontend skeleton.
- `uninstall.php`: Cleanup logic.

## Usage
1. Activate the plugin.
2. Go to **Button Contact** menu in WordPress admin.
3. Configure the contact links and settings.
